Machine Learning Algorithms Cheat Sheet

A comprehensive guide to the most important ML algorithms

Supervised Learning

Algorithms that learn from labeled training data

Linear Regression

Models the relationship between a dependent variable and one or more independent variables using a linear approach.

Type: Regression
Key Concepts: Line of best fit, least squares, RÂ² score
Use Cases: Housing price prediction, stock market analysis

Pros

  • Simple to implement and interpret
  • Computationally efficient
  • Performs well with linear relationships

Cons

  • Assumes linear relationship between variables
  • Sensitive to outliers
  • Prone to underfitting
# Python code example
from sklearn.linear_model import LinearRegression
model = LinearRegression()
model.fit(X_train, y_train)
predictions = model.predict(X_test)

Logistic Regression

Predicts the probability of a binary outcome based on one or more independent variables.

Type: Classification
Key Concepts: Sigmoid function, decision boundary, odds ratio
Use Cases: Spam detection, disease diagnosis

Pros

  • Outputs probabilities
  • Less prone to overfitting
  • Efficient for binary classification

Cons

  • Assumes linear decision boundary
  • Struggles with nonlinear problems
  • Requires feature scaling
# Python code example
from sklearn.linear_model import LogisticRegression
model = LogisticRegression()
model.fit(X_train, y_train)
predictions = model.predict(X_test)

Support Vector Machines (SVM)

Finds the optimal hyperplane that separates classes with the maximum margin.

Type: Classification, Regression
Key Concepts: Kernel trick, support vectors, margin
Use Cases: Image classification, text categorization

Pros

  • Effective in high-dimensional spaces
  • Memory efficient (uses subset of training points)
  • Versatile with different kernel functions

Cons

  • Doesn't perform well with large datasets
  • Sensitive to noise
  • Requires careful parameter tuning
# Python code example
from sklearn import svm
model = svm.SVC(kernel='linear')
model.fit(X_train, y_train)
predictions = model.predict(X_test)

Decision Trees

A tree-like model that makes decisions based on feature values at each node.

Type: Classification, Regression
Key Concepts: Nodes, leaves, entropy, information gain
Use Cases: Customer segmentation, medical diagnosis

Pros

  • Easy to interpret and visualize
  • Handles both numerical and categorical data
  • Requires little data preprocessing

Cons

  • Prone to overfitting
  • Unstable (small changes can lead to different trees)
  • Biased toward features with more levels
# Python code example
from sklearn.tree import DecisionTreeClassifier
model = DecisionTreeClassifier()
model.fit(X_train, y_train)
predictions = model.predict(X_test)

Unsupervised Learning

Algorithms that find patterns in unlabeled data

K-Means Clustering

Partitions data into K distinct clusters based on distance to the centroid of clusters.

Type: Clustering
Key Concepts: Centroids, inertia, elbow method
Use Cases: Customer segmentation, image compression

Pros

  • Simple to implement and interpret
  • Efficient for large datasets
  • Guarantees convergence

Cons

  • Requires specifying number of clusters
  • Sensitive to initial centroids
  • Assumes spherical clusters
# Python code example
from sklearn.cluster import KMeans
kmeans = KMeans(n_clusters=3)
kmeans.fit(X)
labels = kmeans.predict(X)

Principal Component Analysis (PCA)

A dimensionality reduction technique that transforms data to a new coordinate system.

Type: Dimensionality Reduction
Key Concepts: Eigenvectors, eigenvalues, variance
Use Cases: Data visualization, noise reduction

Pros

  • Reduces dimensionality
  • Removes correlated features
  • Improves algorithm performance

Cons

  • Results can be hard to interpret
  • Assumes linear relationships
  • Sensitive to scaling
# Python code example
from sklearn.decomposition import PCA
pca = PCA(n_components=2)
X_pca = pca.fit_transform(X)

Ensemble Methods

Algorithms that combine multiple models to improve performance

Random Forest

An ensemble learning method that constructs a multitude of decision trees at training time.

Type: Classification, Regression
Key Concepts: Bagging, feature randomness
Use Cases: Feature importance, predictive modeling

Pros

  • Reduces overfitting
  • Handles missing values well
  • Provides feature importance

Cons

  • Computationally expensive
  • Less interpretable than single trees
  • Can be slow for real-time prediction
# Python code example
from sklearn.ensemble import RandomForestClassifier
model = RandomForestClassifier(n_estimators=100)
model.fit(X_train, y_train)
predictions = model.predict(X_test)

Gradient Boosting

An ensemble technique that builds models sequentially, with each new model correcting errors made by previous ones.

Type: Classification, Regression
Key Concepts: Boosting, loss function, weak learners
Use Cases: Ranking, predictive modeling

Pros

  • High predictive accuracy
  • Handles mixed data types
  • Supports custom loss functions

Cons

  • Prone to overfitting
  • Computationally expensive
  • Requires careful tuning
# Python code example
from sklearn.ensemble import GradientBoostingClassifier
model = GradientBoostingClassifier()
model.fit(X_train, y_train)
predictions = model.predict(X_test)

Natural Language Processing

Algorithms for processing and understanding human language

Word2Vec

A technique for natural language processing that uses neural networks to learn word associations from large text corpus.

Type: Word Embeddings
Key Concepts: Skip-gram, CBOW, word vectors
Use Cases: Semantic analysis, text classification

Pros

  • Captures semantic relationships between words
  • Works well with large datasets
  • Creates dense vector representations

Cons

  • Computationally intensive
  • Requires large text corpus
  • Cannot handle out-of-vocabulary words
# Python code example
from gensim.models import Word2Vec
model = Word2Vec(sentences, vector_size=100, window=5)
word_vectors = model.wv

BERT

Bidirectional Encoder Representations from Transformers - a transformer-based technique for NLP pre-training.

Type: Language Model
Key Concepts: Transformers, attention mechanism, bidirectional
Use Cases: Question answering, sentiment analysis

Pros

  • State-of-the-art performance on many NLP tasks
  • Understands context bidirectionally
  • Pre-trained models available for fine-tuning

Cons

  • Extremely computationally intensive
  • Requires massive amounts of data
  • Large model size
# Python code example
from transformers import BertTokenizer, BertModel
tokenizer = BertTokenizer.from_pretrained('bert-base-uncased')
model = BertModel.from_pretrained('bert-base-uncased')

Deep Learning

Neural network-based algorithms for complex pattern recognition

Convolutional Neural Networks (CNN)

A class of deep neural networks most commonly applied to analyzing visual imagery.

Type: Neural Network
Key Concepts: Convolution, pooling, filters
Use Cases: Image recognition, object detection

Pros

  • Excellent for image data
  • Automatic feature extraction
  • Parameter sharing reduces complexity

Cons

  • Computationally intensive
  • Requires large amounts of data
  • Black box nature makes interpretation difficult
# Python code example
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Conv2D, MaxPooling2D, Flatten, Dense
model = Sequential()
model.add(Conv2D(32, (3, 3), activation='relu', input_shape=(64, 64, 3)))

Recurrent Neural Networks (RNN)

A class of neural networks where connections between nodes form a directed graph along a temporal sequence.

Type: Neural Network
Key Concepts: Time series, sequences, memory
Use Cases: Time series prediction, speech recognition

Pros

  • Excellent for sequential data
  • Can handle variable length inputs
  • Memory of previous inputs

Cons

  • Vanishing/exploding gradient problem
  • Computationally intensive
  • Slow to train
# Python code example
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import SimpleRNN, Dense
model = Sequential()
model.add(SimpleRNN(50, activation='tanh', input_shape=(10, 1)))
